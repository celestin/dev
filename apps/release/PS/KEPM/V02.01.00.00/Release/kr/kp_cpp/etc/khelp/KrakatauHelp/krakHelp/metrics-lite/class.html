Class Metrics

LOC - Lines of code

Number of Lines in this class, including source, whitespace and comments.

Effectively, a count of the number of newlines in the file.

SLOC - Source lines of code

Number of Source lines in this class, excluding whitespace and comments.

This is not a count of semicolons or distinct statements, but a count of
physical lines that contain source code.

int i=0; float j=0; // This is 1 SLOC

cout << "Testing"
<< " Hello"
<< " there." ; // This is 3 SLOCs
