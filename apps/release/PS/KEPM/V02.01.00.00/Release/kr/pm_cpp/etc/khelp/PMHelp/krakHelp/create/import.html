Import Project

The third method of creating a project in Krakatau is for the Visual Studio user. This method allows the files contained in Visual Studio projects and workspaces to be imported into Krakatau.

create a project using this method, you must select the VS project or workspace by navigating to the file using the file dialog (shown by clicking the "Browse" command). Once the file has been selected choose the "Next" command.

You will see a list of all project configurations in your selected file. Choose which of these are required and then click the "Finish" command to create the projects within Krakatau.
