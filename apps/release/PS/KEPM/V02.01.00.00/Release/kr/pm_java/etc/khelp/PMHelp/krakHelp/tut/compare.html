Comparing Projects

To compare projects follow the steps in the project creation section of the tutorial and create two projects (Usually two version of the same project).

By using the "Select Directory" method of project creation Krakatau automatically sets the "Base Directory" for your projects. The concept of a base directory will be explained fully in a later section of the online help.

  1. Set the Old Project. The first stage of comparing projects is to set one as old, to do this right click on one of your projects and select "Set as old" from the pop-up menu.
  2. Compare Projects. Once a project as been set as old highlight another project and choose the Metrics\Compare Projects command.

Once you have completed these steps click a progress dialog will appear showing that the operation has started.
