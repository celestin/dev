Package Metrics

NFILES - Number of Files

This measures the number of files which comprise a package.  This metric is simply a measure of the size of a package.

NTC - Number of Top-level Classes

The number of top-level classes is indicative of the size of a package.  Also, if we follow the rule of having one top-level class per source file, this number should agree with NFILES.

NTI - Number of Top-level Interfaces

The number of top-level interfaces is a measure of the size of a package.  This value can be used in conjunction with the other package metrics to consider the relative number of interfaces in a package.
