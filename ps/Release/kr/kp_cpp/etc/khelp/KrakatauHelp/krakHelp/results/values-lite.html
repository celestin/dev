Metric Values

The Metric Values screen is split into two panes:

  1. Value table containing Metric names, short codes and values
  2. Help pane, where detailed help on any given metric is provided

The Result Viewer initially shows all metric information for a single symbol. To view the metric information for a particular symbol, select it from the Symbol List.

To get help on a particular metric, click on the row in the table that you would like help about. The help pane will then display a description of the metric, and how it is calculated.
