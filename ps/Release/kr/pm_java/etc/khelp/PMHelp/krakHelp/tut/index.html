Krakatau Tutorial

This Tutorial is meant to familiarize the user with the main functionality of Krakatau. Only the basic features of Krakatau will be discussed here, help for more advanced functionality can found elsewhere in the online help.

This tutorial will take you through the steps involved in creating a project, applying metrics to that project and will briefly describe ways to view the results you have created. The tutorial also shows how to produce an HTML report can be produced to show the results in a web page.
