Creating Projects

To create a project click on the "New Project" command in the File menu. You will then be presented with a dialog box, guiding you through the steps of creating a Krakatau Project. There are three steps to creating a project in Krakatau.

  1. Create a directory for your project. The first stage of this dialog is to give a name to the project that you are creating, to do this you must select a directory from the entry box. Click on the "Browse" button and navigate until you have the desired directory chosen, then click on "Open". You then must decide on a name for you project. Once you have input this information click on the "Next" button of the dialog. You now have a directory prepared for your project.
  2. Choose a project creation method. With the directory prepared, you can now choose the type of project creation that you wish to use. Krakatau offers three methods of project creation: select individual files, a whole directory, or import directly from a Visual Studio workspace (called "Import Project"). For this tutorial choose the "Select Directory" option and click the "Next" button of the dialog. The other methods of project creation will be duscussed elsewhere in the online help.
  3. Select the directory you want to use. You will now be presented with a view that allows you to enter the directory you wish to use. Click on the "Browse" button and choose a directory which contains source code and click "OK".

Once you have completed these steps click on the "Finish" button of the dialog.
