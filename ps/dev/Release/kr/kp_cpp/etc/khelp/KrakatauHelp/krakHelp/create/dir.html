Select Directory

This is the easiest way to create a project, simply select the directory which contains your source files by navigating using the "Browse" command. or by inputting the directory in the entry box.

The "Recursively include sub-folders" check box allows you to choose whether only source files in the selected directory are added or whether source file in sub-directorories are also added.

Note: only appropriate files are added - i.e. source files.
