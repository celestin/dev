File Metrics

LOC - Lines of code

Number of Lines in this file, including source, whitespace and comments.

Effectively, a count of the number of newlines in the file.

SLOC - Source lines of code

Number of Source lines in this file, excluding whitespace and comments.

This is not a count of semicolons or distinct statements, but a count of
physical lines that contain source code.

If you are interested in "logical" lines, see NSC.

int i=0; float j=0; // This is 1 SLOC

cout << "Testing"
<< " Hello"
<< " there." ; // This is 3 SLOCs

NSC - Number of Semicolons

A count of the number of semicolons in this file excluding those within comments and
string literals. This is useful for approximating "logical lines of code". E.g.:

cout << "Hello" << endl ; /* output ; return */
cout << "Hello ; World" << endl ;

NSC = 2.

COM_LOC - Comment lines of code

The total number of comment lines in the file, i.e. C_COM + CPP_COM.

C_COM - C style comments

The number of C style comment lines in the file.

A C style comment begins /* and ends */. E.g.:

int i = 0 ; /* This is a useful
variable here for
many purposes */

C_COM = 3.

CPP_COM - C++ sytle comments

The number of C++ style comment lines in the file.

A C++ style comment begins // until the end of the line. E.g.:

int i = 0 ; // Index variable
float pi = 3.14 ; // Handy pi variable
char star = '*' ;

CPP_COM = 2.

COM_RAT - Comments Ratio

The ratio, for this filel, of comment lines to lines of code, i.e. COM_LOC / LOC.

TCOM_RAT - True Comments Ratio

The ratio, for this filel, of comment lines to source lines of code, i.e. COM_LOC / SLOC.
