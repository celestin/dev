LCOM - Lack of Cohesion Methods

Formula Σ

This metric quantifies how much intercourse the methods of a class have
with the member variables of that class.

LOCM is calculated by building a list of member variables and a count
of the number of references to each variable (usage) in all methods
of this class.

Then, a sum of the ratios of usage / TotalMethods is calculated.

Finally, LOCM is returned as this SumOfRatios / TotalAttributes.

This is not the original calculation proposed by Chidamber &amp; Kemerer [CHI94],
but a revised version by Henderson-Sellers [HEN96].
See http://www.computing.dcu.ie/~renaat/ca421/LCOM.html for more information.
