Trouble Shooting

The following is a list of issues that should be referred to before contacting Power Software:

  • Clicking "Apply Metrics" has no effect

    • Make sure that a project in the project tree is selected (highlighted).

  • After sorting the symbol list the application freezes

    • For a large project sorting may take some time - once the sort has been completed the user interface will respond

These are the only currently known issues, if you find something other than this please email the information to krakatau@powersoftware.com
