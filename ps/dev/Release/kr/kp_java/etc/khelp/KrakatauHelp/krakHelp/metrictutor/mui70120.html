NEST - Maximum number of levels

NEST indicates the maxium number of nests of like constructs with the method.

for (int i=0; i<5; i++)
&nbsp;if(i<2)
&nbsp;&nbsp;if(i>0)
&nbsp;&nbsp;&nbsp;j=3 ;
&nbsp;&nbsp;else
&nbsp;&nbsp;&nbsp;j=4 ;
&nbsp;else
&nbsp;&nbsp;j++ ;

NEST = 2 because there is an if within an if.
