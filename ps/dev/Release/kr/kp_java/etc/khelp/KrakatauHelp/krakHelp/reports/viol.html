A Violating Report will display which metrics have exceeded bounds you have selected. Before you create a Violating Report, first ensure that Bounds have been selected for your file. To create a Violating Report, click on the "Violating Report" command in the Reports menu. You will then be presented with a dialog box. Use the browser to select the intended directory, and give your report a name.

Please note to create a Violating Report, first ensure that you have entered Bounds for your file. For further information on bounds, please see the Metric Preferences section, under Choosing Metrics.
