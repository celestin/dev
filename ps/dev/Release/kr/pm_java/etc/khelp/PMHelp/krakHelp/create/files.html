Select Files

The "By Files" project creation allows you to specify which exact files you wish to add to your project.

This method allows you to select files from your directory structure. The arrow buttons in the dialog have the following functionality:

Add the highlighted file or directory from the left list to the right.
Recursively add all files from the current directory.
Remove the selected file(s) from the right list.
Remove all the files from the right list.

Note only files of the appropriate types will be added to the project, also when an entire directory is added all of the appropriate files in that directory and any sub-directories will be added as well.
