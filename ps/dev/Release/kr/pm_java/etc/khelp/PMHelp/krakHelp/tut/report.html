HTML Report

This tutorial creates a HTML report of metric values created by Krakatau.

To do this, in the Result Viewer window, click on the HTML Report command in the Reports menu. You will then be presented with a file dialog box. Navigate to the directory where you wish the Report to be created and click "Open".

It should be noted that due to the nature of the HTML report with large project there may be many HTML files created in the directory that you choose - therefore it might be advisable to create a directory especially for the HTML report.
