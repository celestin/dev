Comparison Values

There are three tabs in the result window, each of these have different types of metrics relating to the selected symbol.

  1. Changed metrics relate to additions/deletions and changes since the previous version of this symbol.
  2. Difference metrics show the difference between metrics values since the previous version of this symbol. For example +10 LOC if the lines of code has increased by 10.
  3. Actual metrics give the metric values for this symbol (regardless of the previous metrics).
