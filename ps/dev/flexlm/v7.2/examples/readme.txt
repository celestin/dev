THIS DIRECTORY CONTAINS UNSUPPORTED PROGRAMS, AND THESE PROGRAMS MAY NOT
EXIST ON FUTURE FLEXlm DISTRIBUTION KITS.

This directory contains several example program directories.  These programs
are intended to provide instruction in the use of various FLEXlm features.

While we have made every effort to insure that these examples are correct,
they are not intended to be release-quality debugged products.  In other
words, USE THESE EXAMPLES AT YOUR OWN RISK!  

The examples provided include:
	
	api- examples of console apps using trival and simple api

	win_api - examples of windows apps using trivial and simple apps

	filter - Vendor daemon checkout filter example.

	vendor_hostid - Example of vendor-defined hostid code

	licenses -- examples licenses, useful for lmcrypter

	ls_co_ex.c -- example of ls_checkout -- vendor daemon checkout filter.

	vendor_info -- example of using lc_get_attr(job, LM_A_VD_FEATINFO...).
