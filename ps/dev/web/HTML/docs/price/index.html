Power Software Licensing

Power Software offers two forms of licensing:

  1. Host-locked (also “Node-locked”) whereby the machine name is locked into the license and the software can only run on that machine.
  2. Floating whereby the software can be installed on an unlimited number of machines (provided they can access the server) but only one user can use the software concurrently.
