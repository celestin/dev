Krakatau Essential PM is our flagship graphical metrics tool.

KEPM is more than a graphical front-end to EPM, it is an essential component of the CMVC professional's toolkit.

All the metrics provided by our popular Essential Project Manager (EPM) tool including the unique Changed SLOC metrics are offered by KEPM, as well as the 'differences' in the metrics between releases.

Essential Project Manager is our latest, command line metrics tool.

EPM combines the simplicity and elegance of our popular Essential Metrics tool with the unique Changed SLOC metrics of our market-leading Project Manager products.

Designed for use by CMVC (including Clearcase) professionals as well as software quality devotees, EPM is a great new edition to our portfolio designed to improve and protect your source code investment.


Essential Metrics is an advanced command line metrics tool for C/C++ and Java software projects.

Invoked directly at the command line, Essential Metrics delivers numerous popular metrics and allows these to be reported using CSV, HTML and XML.

Essential Metrics is ideal for inclusion in your batch/build process for automated metric results.

View the Flash presentation at www.powersoftware.com/em/.

Krakatau Project Manager is the only source code metrics tool that compares two C/C++ or Java projects and shows the 'changed, deleted and added SLOC' between the projects. These changed metrics give essential information about developer activity and can be used to pinpoint problem areas in your source code projects.

Krakatau Project Manager was designed and specified by our customers and is ideal for use with leading CMVC tools like ClearCase from Rational Corporation.

The CHG_SLOC, ADD_SLOC and DEL_SLOC metrics are unique to Power Software and Krakatau PM.

Krakatau Professional is the most comprehensive, commercially supported source code metrics tool for C/C++ and Java projects.

Krakatau Professional is scalable to millions of source lines of code and has over 70 source code measurements including Halstead size metrics, complexity and advanced object-oriented metrics.

Metrics can be collected at the command line or using our specially designed GUI featuring histograms, kiviat diagrams and numerical reports.

Violating metrics reports can be created by setting upper and lower bounds and all metrics can be sorted and selected/deselected.

Output is to html (for intranets) or CDF/CSV (for spreadsheets or PC anaylsis applications).

Krakatau Professional is aimed at the Class, Method and Interface level metrics.

Visualize it! is a graphical source code visualization tool for Java.

Visualize it! is scalable, affordable and intuitive. Visualize it! displays all meaningful relationships between Java source code symbols in a variety of useful 2D and 3D views.

Visualize it! includes handy rollups, for Symbol Properties and Symbol Search.

Visualize it! is scalable to many millions of Java source lines and includes FLEXlm License Management.
