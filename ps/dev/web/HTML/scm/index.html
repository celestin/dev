Purpose

SourceCodeMetrics.com has been started to establish a collaboration area for all forms of source code (software) metrics, to enable people of varying roles to grasp the benefits, current trends, and best practices for implementing source code metrics systems.

We aim to help you understand how measuring source code metrics can:

  • improve software maintainability
  • gauge progress
  • optimize testing

Commercially-available Tools

The following tools are available under a variety of licensing options to enable you to integrate the measuring and reporting of source code metrics into the build process:
