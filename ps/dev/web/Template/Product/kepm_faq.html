How do I install KEPM on Vista or Windows 7?

Simple! Just ensure that you right-click on the KrakatauEpmSetup.exe and choose Run as administrator.

Why? If you don't, MySQL won't be properly registered and the service won't be started.

When I analyse, KEPM says the database is not started... why?

This is likely to be caused by a faulty installation. Try uninstalling and reinstalling, and if that doesn't work, ensure the EPMdb Service is started. If not, you can start it manually.
