Histogram View

The histogram shows a graphical view of the differences between the values of a particular metric over several symbols.

To view the histogram click on the "Histogram" tab under the Results Pane. You will then be shown a bar graph - with the currently selected symbol(s). To compare the metric value over several symbols simply select several items in the symbol list. Note: to select multiple items in the list use the standard <Shift> and <Ctrl> buttons along with the mouse buttons.

To change the currently displayed metric, select another metric from the pull down list above the histogram.

The histogram view is only available on the File, Class, and Method tabs.
