Selecting Symbols

Use the Symbol list in the Metric Result Viewer to choose the symbols you wish to examine. Select a symbol by clicking on it - this will update the Result Viewer to show information concerning that symbol.

To select multiple symbols use the <Ctrl> + mouse click and <Shift> + mouse click commands. When multiple symbols are selected the last one selected is highlighted in blue. Any single-metric information you have selected will pertain to this highlighted symbol.
