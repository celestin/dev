Kiviat View

The Kiviat view, which can be accessed by the Kiviat tab, shows the average for metrics values over selected symbols.

Note: before the Kiviat View can be used bounds have to be set in the Bound Editor.

The following is an example of a Kiviat:

The inner circle signifies the lower bound for each metric, the outer circle signifies the upper bound. Therefore, all metrics values which are between the inner and outer circles (shown green above) are not violating bounds.

If only one symbol (from the Symbol list) is selected then the Kiviat Diagram shows which values are violating bounds for that symbol. When more than one symbol is selected then the nodes in the Kiviat Diagram are averages for the selected symbols.
