Note 2 Operating Systems Requirements for Java Runtime

There are several Operating System patches for the Java Runtime system which you may need to install.

These are as follows:

SUN Solaris Sparc

Patches may be required for the Java runtime environment on the SUN Sparc Solaris operating systems. You can use the:

showrev -p

command to check which patches are currently installed on your operating system. For more information, see http://www.sun.com/solaris/java

HP-UX 10.20

On HP-UX 10.20 systems you might need some operating system patches for Java runtime.
