Command-line operation
Essential Metrics can be easily operated from the command line. Quick and simple to include in your automated processes!

CSV Report
The CSV report can be easily opened with a spreadsheet application like Microsoft Excel. From there you can do futher analysis and graphing.

CSV Report - Line graph
A line graph can be easily created to show high metric values. This line graph shows LOC & SLOC for Files.

CSV Report - Bar chart
A stacked bar chart can show related metrics very visually. This bar chart shows Cyclomatic Complexity and Source Lines of Code for certain Methods.

CSV Report - Pie chart
A pie is useful for looking at one metric across many symbols. Here, the CSAO (Class Size) for various classes is shown in a 3D pie chart.
