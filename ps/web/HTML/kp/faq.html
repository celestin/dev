  • How do I create a project?

    • How do I create a project? 

    Click on the File/New command and then follow the instructions of the New Project dialog. Firstly you will have to choose a directory for the project to be located and a name for the project. By clicking on the "Next" command you will then be given a choice of 3 project creation types. To get started choose the "Directory" type and click next. You will then be prompted to input the directory that contains your source files. Once you have clicked "Finish" a new project will be created with all the source files from the directory that you chose.
